# **App Name**: PortfolAI

## Core Features:

- AI Theme Generation: Use generative AI to propose an appropriate design theme based on user's profession.
- UI Customization: Enable users to preview portfolio website with AI-generated theme and adjust basic UI elements such as layout and colors.
- Deployment: Support direct deployment to platforms like Vercel or Netlify, requiring the user to authenticate and grant necessary permissions.

## Style Guidelines:

- Primary color: A soft blue (#77B5FE) evoking trust, professionalism, and innovation, aligning with the goal of aiding users in career advancement.
- Background color: A very light gray (#F0F4F8), providing a clean, neutral backdrop that ensures readability and visual clarity, fitting a light color scheme.
- Accent color: A muted purple (#B19CD9) providing a touch of creativity without being distracting.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look, well-suited to professional content.
- Use a set of simple line icons from a public set such as Tabler Icons, in the primary color, to represent key sections and interactive elements, ensuring a consistent and modern visual language.
- Implement a clean, single-page layout, ensuring easy navigation and optimal viewing across devices.
- Add subtle transitions on UI elements and interactive components (buttons, links), using short delays, to enhance the user experience.
